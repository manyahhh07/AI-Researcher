If your goal is to **impress recruiters**, avoid projects like Virtual Mouse, Air Canvas, Finger Counter, Fruit Ninja, Piano, Virtual Keyboard, Face Filters, etc. Thousands of GitHub repos already have those.

The projects that go viral now combine **computer vision + physics + generative visuals + interaction**. Think of them as interactive digital art rather than CV demos.

## Tech Stack That Makes These Stand Out

* **MediaPipe Hands** or **MediaPipe Tasks** for hand/body tracking
* **OpenCV** for webcam capture and image processing
* **Three.js** or **Babylon.js** for GPU-accelerated 3D rendering
* **Matter.js** or **Rapier** for realistic physics
* **WebGL shaders (GLSL)** for portals, fluids, and energy effects
* **Tone.js** for gesture-driven audio
* **TensorFlow.js** (optional) for custom gesture or expression recognition
